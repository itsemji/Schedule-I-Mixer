Why Some Effect Combinations Don't Work

If you're trying to reverse engineer a specific chain and not getting results, don't worry — you're not doing anything wrong. Some combinations of effects are either logically impossible or extremely unlikely to occur due to how ingredients interact.

How Contradictions Happen

  • Rule Conflicts: Some effects overwrite others during mixing (e.g., Energizing → Schizophrenia).
  • Self-Canceling Chains: Certain ingredients reverse each other's changes.
  • Unreachable Effects: Some effects can only be reached via very specific sequences — or not at all.

Examples of Contradictory Combinations

  • Energizing + Schizophrenia: Motor Oil replaces Energizing with Schizophrenia.

  • Munchies + Schizophrenia: Same issue — only one survives depending on conditions.

  • Euphoric + Sedating: Ingredients tend to convert Euphoric to something else before Sedating applies.

  • Focused + Seizure-Inducing: Banana transforms Focused into Seizure-Inducing — so they can’t coexist.

  • Smelly + Anti-Gravity: Banana transforms Smelly into Anti-Gravity, removing the original effect.

  • Disorienting + Electrifying: One tends to cancel or overwrite the other in chains.

What You Can Do

  • Try using fewer or different effects if no chain is found.
  • Refer to the Calculator page to experiment with ingredient logic directly.
  • If you're still stuck, check the Help page or let us know!
